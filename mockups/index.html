칸반 보드 — 디자인 시안 비교 (5안)

미니멀·차분 톤을 공통 기준으로, 레퍼런스 앱 5개의 레이아웃 구조를 각각 살렸습니다. 동일한 샘플 데이터(팀원 5명·카드 8장)로 비교합니다.

각 시안을 스크롤해 살펴본 뒤, 아래로 내려가며 A~E를 비교해보세요. 마음에 드는 하나를 고르시거나 하이브리드(예: “C 레이아웃 + E 카드”)도 가능합니다.

A · Trello B · Linear C · Notion D · Jira E · Height
시안레이아웃 구조분위기강점약점잘 맞는 사람
A · Trello 상단 헤더 + 가로 3컬럼
(사이드바 없음)		 부드러운 종이 카드 상태를 공간으로 즉시 파악, 담당자 가시성 강함 카드 적으면 가로 공간 낭비 매일 훑는 PM (지훈)
B · Linear 좁은 좌측 사이드바 + 고밀도 보드 거의 모노톤 + 단일 액센트 정보 밀도 최고, 스캔 빠름, 키보드 친화 촘촘한 타입은 터치에서 답답할 수 있음 속도 중시 파워유저
C · Notion 넓은 좌측 네비 + 문서형 보드 따뜻한 종이 뉴트럴, 에디토리얼 차분·여백, 읽기 좋음, 낮은 시각 소음 넓은 사이드바가 보드 공간 잠식 차분함 선호 (수민)
D · Jira 상단바 + 아이콘 네비 + 고밀도 카드 프로페셔널 슬레이트·블루 메타데이터 풍부(이슈키·뱃지·WIP), 책임 명확 가장 밀도 높아 스캔 부담, 아이콘 네비 학습 필요 스프린트 운영 리드 (지훈·관리)
E · Height 뷰 스위처 탭 + 필터 바 + 모던 카드 프레시 틸 액센트, 세련된 SaaS 깔끔·경쾌, 필터/정렬 접근 좋음 좌측 네비 없어 깊은 탐색 제약, 팔레트가 다소 범용적 가벼운 모던 도구 선호

시안 A · Trello 스타일 ↗ 새 탭에서 크게 보기

가로 3컬럼 보드 · 종이 카드가 떠 있는 구조. 상태를 한눈에 훑기 좋음.

시안 B · Linear 스타일 ↗ 새 탭에서 크게 보기

좁은 사이드바 + 고밀도 보드 · 거의 모노톤에 급한 것/진척만 색을 씀.

시안 C · Notion 스타일 ↗ 새 탭에서 크게 보기

넓은 사이드바 + 문서형 보드 · 따뜻한 뉴트럴, 여백 넉넉한 블록 카드.

시안 D · Jira 스타일 ↗ 새 탭에서 크게 보기

상단바 + 아이콘 네비 + 고밀도 카드 · 이슈키·우선순위 뱃지·WIP 제한 표시.

시안 E · Height 스타일 ↗ 새 탭에서 크게 보기

뷰 스위처 탭 + 필터 바 + 모던 카드 · 프레시 틸 액센트의 세련된 SaaS 톤.
