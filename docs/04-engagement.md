# 04. 차별화·성취감 (Engagement)

비슷한 칸반 앱 사이에서 이 앱을 계속 쓰게 만드는 장치.
채택 5개(E2·E3·E4·E6·E7)를 MVP에 넣고, 나머지는 [나중에 추가](#나중에-추가-후보-기능)로 관리한다.

---

## 채택 기능

### E2. 오늘의 진척 바 (성취감)
- 상단에 "오늘 완료 N / 전체 M" 진행률 바 표시.
- **기준**: 오늘 Done으로 이동한 카드 수 / (오늘 기준 활성 카드 총합).
- **효과**: 하루 단위 성취를 눈에 보이게 해 재방문 유도(지훈·수민).

### E3. Done 카운터 · 스트릭 (성취감)
- 누적 완료 카드 수 + **연속으로 카드를 완료한 날(스트릭)** 표시.
- **효과**: "오늘도 이어가자"는 동기. 개인 학습용 앱의 지속 사용 장치.
- **주의**: 스트릭이 압박이 되지 않게 부드러운 톤(끊겨도 비난하지 않음).

### E4. 담당자 색상 시스템 (인지 부담 감소, 원칙 P3 강화)
- 팀원마다 **고유 색 + 이니셜** 부여. 카드에 담당자 칩으로 표시.
- 미배정 카드는 회색/점선 등으로 구분.
- **효과**: "내 것"이 즉시 보이고(수민·태오), 미배정 일이 드러남(엣지 대응).

### E6. 컬럼 카운트 & WIP 힌트 (인지 부담 감소, 원칙 P7 강화)
- 컬럼 헤더에 카드 수 표시. **Doing이 일정 수 이상이면** 부드러운 경고 신호(색/문구).
- **효과**: 병목·과부하가 한눈에(지훈). WIP 제한 문화의 가벼운 버전.

### E7. 빠른 필터 + 정렬 선택 (인지 부담 감소, 원칙 P2·P6 강화)
- **필터(원클릭 토글)**: "내 카드만" / "이번 주 마감" / "High만".
- **정렬 선택** — 사용자가 고를 수 있음:
  - **내 것 우선** — 담당자가 나(현재 선택된 팀원)인 카드를 위로.
  - **급한 것 우선** — 마감 임박·초과 + High 우선순위를 위로.
  - (기본: 수동 정렬 순서 유지)
- **효과**: 태오는 "내 카드만"으로 바로 도달, 지훈은 "급한 것 우선"으로 리스크 점검.

## 시나리오에 미치는 영향

| 기능 | 영향받는 시나리오 | 어떻게 |
|---|---|---|
| E2 진척 바 | S5(퇴근 후 확인) | "오늘 얼마나 됐나"를 즉시 확인 |
| E3 스트릭 | 전체 재방문 | 매일 여는 동기 부여 |
| E4 담당자 색 | S3·S6(내 카드 식별) | 색/이니셜로 즉시 식별, 미배정 구분 |
| E6 컬럼 카운트 | S4·S5(리스크·진척) | Doing 과부하·컬럼별 현황 노출 |
| E7 필터·정렬 | S4·S6(리스크·태오 확인) | 급한 것/내 것 우선으로 정렬·필터 |

## 의도적으로 넣지 않는 것 (MVP)

- **완료 애니메이션(E1)** — 재미 요소지만 없어도 핵심 성취(E2·E3)로 충분. 과한 연출 배제.
- **강한 알림/푸시** — 개인 학습용이라 방해 요소 최소화. 스트릭도 압박형이 아니라 격려형.
- **게임화 과잉**(배지·레벨·포인트 상점 등) — 학습용 스코프에 과함.

## 나중에 추가 후보 기능

> 이번 MVP에는 넣지 않지만, 확장 시 우선 검토할 목록.

| 후보 | 내용 | 비고 |
|---|---|---|
| E1 완료 축하 효과 | Done 이동 시 애니메이션/반짝임 | 성취감 강화, 저비용 |
| E5 긴급 카드 자동 강조 | 마감 임박·초과·High 자동 색/뱃지 | *일부는 E6/E7 정렬로 커버되나, 카드 자체 강조는 별도* |
| E8 즉석 카드 입력 | 컬럼 상단 엔터 연속 생성 | 원칙 P1 강화 |
| E9 키보드 단축키 | n/화살표 등 | 파워유저용 |
| E10 마감 퀵버튼 | 오늘/내일/주말 버튼 | 입력 속도 |
| E11 마감 임박 정렬 뷰 | 급한 순 보조 뷰 | E7 정렬과 중복 검토 |
| E12 빈 상태 일러스트 | 빈 보드/컬럼 안내 | 첫 사용 이탈 방지 |
| E13 미배정 알림 배너 | "담당자 없는 카드 N개" | 엣지 대응 |
| E14 마감 지난 카드 모아보기 | 방치 카드 모음 | 엣지 대응 |
