# 05. 디자인 (Design) — 확정본

**컨셉: 5가지 테마 전환형 칸반 — 파스텔 4종 + 다크모드**

기본 틀은 정보 밀도 있는 프로 도구형(상단 앱바 + 아이콘 네비 + 툴바 + 필터/정렬 바 + 3컬럼)이되,
컬럼별 회색 배경 박스는 없애 카드가 배경 위에 바로 놓이는 가벼운 구조.
전체 색은 **CSS 변수(토큰)** 로 구동해 5가지 테마를 하나의 마크업이 공유한다.

- **기본 테마: A · Periwinkle & Mint**
- 확정 프로토타입: `mockups/concept-themed.html` (테마 스와치로 실시간 전환, 선택은 localStorage 저장)

---

## 레이아웃 구조

```
┌────────────────────────────────────────────────────────────┐
│ APPBAR: 로고 · 보드 스위처       [테마 스와치 ●●●●■] · 내 아바타 │
├──────┬─────────────────────────────────────────────────────┤
│ ICON │ TOOLBAR: 브레드크럼 · 보드 제목 · 🔥스트릭 · 진척바 · 아바타 │
│ NAV  ├─────────────────────────────────────────────────────┤
│ ▦ ☰  │ FILTER BAR: [내 카드만][이번 주][High만]      정렬 ▾    │
│ ⟳ 📊 ├─────────────────────────────────────────────────────┤
│ 👥   │  To Do (3)      Doing (3) ⚠      Done (2)            │
│      │  ┌ 카드 ┐        ┌ 카드 ┐         ┌ 카드 ┐            │
│ ⚙    │  └─────┘        └─────┘          └─────┘            │
│      │  ＋ 카드 추가     ...              ...                 │
└──────┴─────────────────────────────────────────────────────┘
```

- **앱바**: 로고/보드 스위처 + **테마 선택 스와치 5개** + 현재 사용자 아바타.
- **아이콘 네비(56px)**: 보드/백로그/스프린트/리포트/구성원/설정 (MVP는 보드 중심, 나머지는 자리만).
- **툴바**: 브레드크럼 + 보드 제목 + 진척 바(E2) + 🔥 스트릭(E3) + 멤버 아바타 스택.
- **필터 바**: 내 카드만/이번 주 마감/High만(E7) + 정렬(수동/내 것 우선/급한 것 우선, E7).
- **보드**: 3컬럼 고정, **컬럼 배경 박스 없음**. 컬럼 헤더에 카드 수, Doing 과다 시 WIP 경고(E6).
- 모바일(<700px): 아이콘 네비가 상단 가로 바로, 컬럼은 세로 스택.

## 테마 시스템 (핵심)

5개 테마를 `[data-theme]` 속성 + CSS 변수로 전환. **기본값 = `a`**. 선택값은 localStorage(`kanban-theme`)에 저장.

| 테마 | key | 성격 |
|---|---|---|
| **Periwinkle & Mint** (기본) | `a` | 시원한 파스텔 |
| Peach & Sage | `b` | 따뜻한 파스텔 |
| Lavender & Sky | `c` | 라벤더 드리미 |
| Dusty Muted | `d` | 차분한 뮤트 |
| Dark Mode | `dark` | 다크 |

### 토큰 목록 (테마마다 값이 바뀌는 변수)

- 표면/배경: `--bg` `--surface` `--appbar` `--field`
- 텍스트: `--ink` `--ink-soft` `--ink-mute`
- 경계: `--line` `--line-soft`
- 컬럼 액센트: `--todo` `--doing` `--done`
- 우선순위 칩: `--high-bg/--high-fg` `--med-bg/--med-fg` `--low-bg/--low-fg`
- 마감 경고(임박/오늘): `--warn-bg/--warn-fg`
- 네비/활성: `--navbg` `--active-bg/--active-fg/--active-border`
- 그림자/라운드: `--shadow` `--shadow-lift` `--card-radius`

## 컴포넌트 명세

### 카드 (요약 겉면 — 원칙 P5)
- 상단: 타입 색점 · 이슈키(예: CORE-14) · 우선순위 칩(▲High/■Med/▼Low, `--high/med/low`).
- 제목(1줄, 말줄임).
- 하단: 마감(임박/오늘=`--warn` 강조, 없으면 "마감 없음") · 담당자 아바타(색+이니셜).
- **미배정**: 점선 아바타 + "미배정" 태그(원칙 P3).
- **Done 카드**: 점선 테두리 + 완료 표시, 톤 다운.

### 카드 상세 모달 (편집 — MVP)
제목 · 설명(메모) · 담당자 · 마감일 · 우선순위 · [삭제]. 정밀 편집 전용(P1/P5).

### 컬럼
헤더 = 액센트 점 + 이름 + 카드 수. Doing 임계 초과 시 `--warn` WIP 경고(E6/P7). 하단 "+ 카드 추가"(제목만 즉시 생성, P1).

### 진척 바·스트릭·필터·정렬
- 진척 바: "오늘 완료 N/8", 채움 = `--done` (E2).
- 스트릭: "🔥 N일 연속" (E3, 격려 톤).
- 필터 칩 3종 + 정렬 셀렉트(수동/내 것 우선/급한 것 우선) (E7).

### 테마 스와치
앱바 우측 5개 버튼. 각 스와치는 해당 테마의 대표 3색 그라디언트. 클릭 시 `data-theme` 교체 + `aria-pressed` 갱신 + localStorage 저장.

## 인터랙션

- **드래그 앤 드롭**: 컬럼 간 이동 + 같은 컬럼 내 순서 변경(P4). 드롭 인디케이터.
- **모바일 대안 이동**: 드래그 어려운 상황 대비 카드 단계 이동 컨트롤(P2).
- **즉시 생성**: 컬럼 하단 입력 → Enter 연속 생성(P1).
- **카드 클릭**: 상세 모달.
- **필터/정렬/테마**: 즉시 반영, 활성 상태 시각 표시.
- **완료 이동**: 과한 애니메이션 없이 잔잔히(E1 제외).

## 타이포그래피

- 시스템 humanist 스택(`-apple-system, "Malgun Gothic", "Apple SD Gothic Neo", sans-serif`).
- 스케일(예): 보드 제목 20 / 카드 제목 13.5 / 메타·라벨 11~12. 라벨은 normal-case, 과한 대문자·자간 지양.

## 반응형

| 구간 | 레이아웃 |
|---|---|
| ≥1024px | 아이콘 네비 + 3컬럼 가로. |
| 700–1023px | 3컬럼 유지(가로 스크롤 허용). |
| <700px | 아이콘 네비 상단 가로 바, 컬럼 세로 스택, 카드 단계 이동 컨트롤 노출, 테마 라벨 숨김. |

## 접근성

- 대비: 각 테마 텍스트/배경 WCAG AA 목표(다크 포함). 색만으로 의미 전달 금지(우선순위·마감은 색+아이콘/텍스트 병행).
- 포커스: `--active-fg` 포커스 링.
- 드래그 대안 조작 제공.
- 한글 IME: 조합 중 손실/중복 없이 입력.
- `prefers-reduced-motion` 존중(테마 전환·hover 트랜지션 축소).

## 이 단계에서 확정하지 않은 것

- 폰트 임베드 방식, 아이콘 세트 → 구현 시 결정.
- 다크모드 자동(OS 연동) 여부 → 우선 수동 선택. OS 연동은 확장 후보.
- E1 완료 애니메이션 등은 확장 후보(04 문서).
